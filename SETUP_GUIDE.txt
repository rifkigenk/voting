STEP 1: CREATE DATABASE
-----------------------
1. Open phpMyAdmin (http://localhost/phpmyadmin)
2. Create a new database named: mps_voting
3. Charset: utf8_unicode_ci

STEP 2: RUN DATABASE SETUP
---------------------------
1. Open browser: http://localhost/Projek%20MPS/mps-voting/php/database.php
2. This will automatically create all necessary tables:
   - voters
   - candidates
   - votes
   - admin

STEP 3: CREATE ADMIN ACCOUNT
-----------------------------
1. Open phpMyAdmin ‚Üí mps_voting ‚Üí admin table
2. Add new record:
   - username: admin
   - password: (use password hash below)
   - email: admin@school.com

3. Generate password hash using PHP:
   <?php echo password_hash('password123', PASSWORD_DEFAULT); ?>
   
   Use this hash in the password field

ALTERNATIVE: Run this SQL query directly:
INSERT INTO admin (username, password, email) VALUES 
('admin', '$2y$10$XW8LlY6z1e5j8e5j8e5j8e5j8e5j8e5j8', 'admin@school.com');

DATABASE STRUCTURE
------------------
voters Table:
- voter_id (PK)
- nisn (UNIQUE)
- name
- class
- email
- phone
- has_voted (0/1)
- created_at
- updated_at

candidates Table:
- candidate_id (PK)
- name
- position
- class
- photo
- bio
- votes (count)
- created_at
- updated_at

votes Table:
- vote_id (PK)
- voter_id (FK)
- candidate_id (FK)
- position
- voted_at

admin Table:
- admin_id (PK)
- username (UNIQUE)
- password (hashed)
- email
- created_at

IMPORTANT NOTES
---------------
1. Each voter can only vote once
2. Passwords are hashed with bcrypt for security
3. Data is stored in MySQL database
4. Application uses prepared statements to prevent SQL injection
5. Session management for voter and admin authentication
6. Real-time result updates
